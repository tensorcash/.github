TensorCash is an independent layer-1 where the work that secures the chain is *useful*:
miners run verifiable AI inference and the network cryptographically verifies the results.

### Repositories

- **[tensorcash](https://github.com/tensorcash/tensorcash)** — umbrella: node, miner & verification services
- **[bcore](https://github.com/tensorcash/bcore)** — consensus node (assets, models, VDF-SPV, post-quantum, ZK-KYC)
- **[vllm](https://github.com/tensorcash/vllm)** · **[llama.cpp](https://github.com/tensorcash/llama.cpp)** — inference engines with the proof-of-inference sampler
- **[vllm-production-stack](https://github.com/tensorcash/vllm-production-stack)** — serving stack
- **[gnark](https://github.com/tensorcash/gnark)** — zero-knowledge proving

### Get involved

We welcome contributors — both as maintainers and as supporters of the project's infrastructure:

- **Code &amp; maintainers** — open an issue or pull request in any repository above. Consensus-affecting changes receive extra review.
- **Build infrastructure** — if you can provide CI/build **runners** or help with macOS/Windows **code-signing**, [get in touch](mailto:contact@tensorcash.org) and we'll coordinate directly.
